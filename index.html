Текстовая РПГ

github.com/ElisabethMesh/ConsoleGame

Данное приложение построено на Apache Maven и имеет подключение к MySQL.

Для начала, выберите себе героя и вступите в клан. Цель игры - заработать как можно больше золота для своего клана, выполняя разные действия. Все операции с золотом хранятся в базе данных.

Конкурс ресторанов

github.com/ElisabethMesh/restaurant_competition

Приложение написано с применением Spring Frameworks и MySQL, покрыто тестами Junit5.

Есть две роли: администратор и пользователь. Только администратор может добавлять рестораны и управлять ими, включая изменение меню ресторанов. Пользователь может только управлять своей учетной записью и голосовать за любимый ресторан.
